CHANNELS

Features required
 - Get all channels on the platform
 - Get a channel

Not required
 - Adding channels
 - Updating channels
 - Deleting channels


USERS

Features required (call through Auth0 API)
 - Getting a user
 - Updating a user
 - Deleting a user

Features required (non-Auth0 related)
 - Get a user's subscriptions

Not required (handled by Auth0 only)
 - Creating a user (handled on sign-up)


SUBSCRIPTIONS

Features required
 - Get a subscription
 - Add a subscription (also adds channel)
 - Delete a subscription (also deletes channel as required)

Not required
 - Update a subscription
 - Get all subscriptions on the platform
